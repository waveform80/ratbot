This page lists the software used in the creation of this site, as a little thank you to the various open-source projects that helped make it possible. Firstly the software used by Holly for drawing:

  • Inkscape - a vector painting package
  • GIMP - a bitmap paint package, for touching up photos

Here's the software Dave used to construct and host the website:

  • Apache HTTPD web server
  • PostgreSQL database server
  • TurboGears web framework, consisting of:
    • SQLAlchemy ORM
    • Paster middleware
    • Genshi template library
    • ToscaWidgets forms library
  • Which is all built in the awesome Python scripting language

Everything runs on Ubuntu Linux and the server is an Amazon EC2 instance.
